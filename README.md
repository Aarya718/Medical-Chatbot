## SAFESPACE AI AGENT

An AI-powered emotional support assistant designed to provide real-time mental health guidance and crisis detection using Large Language Models and tool-based agent architecture.

## 📌 Project Overview

SafeSpace AI Agent is an intelligent conversational system built to provide 24/7 emotional support. 
The system analyzes user input, detects emotional distress, and can trigger emergency alert mechanisms when high-risk situations are identified.

The project demonstrates the use of AI agents, tool invocation, and real-time conversational workflows.
